## Per-interview checklist

1. [ ] Send pre-interview form + consent 24h before call
2. [ ] Confirm recording consent at start of call
3. [ ] Follow [interview-guide-en.md](../../interviews/guides/interview-guide-en.md) (or VI/FI)
4. [ ] Run demo per [demo-script.md](../../specs/demo-script.md)
5. [ ] Show [mock-social-post.json](../../../../samples/market-validation/mock-social-post.json)
6. [ ] Complete [post-interview.md](../../interviews/questionnaires/post-interview.md) immediately after
7. [ ] Write summary using [interview-summary-template.md](../../interviews/templates/interview-summary-template.md)
8. [ ] Upload recording; add link to recruitment tracker
9. [ ] Update participant status in [README.md](../../README.md)
