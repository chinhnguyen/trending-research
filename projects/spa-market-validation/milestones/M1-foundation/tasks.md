## Pre-demo checklist {#pre-demo-checklist}

Before first interview, confirm:

- [ ] Web app runs locally (`willbe-api` + `npm run dev` in `web/`)
- [ ] API keys configured in `.env`
- [ ] 2–3 saved reports per region (VN, FI, global) as fallback if live demo fails
- [ ] Mock social post JSON open in browser or printed
- [ ] Zoom/Loom tested with recording + consent language ready
- [ ] Team has rehearsed 45-min flow end-to-end

## Exit criteria

Team can run a 45-minute interview end-to-end with demo + mock content cards.
