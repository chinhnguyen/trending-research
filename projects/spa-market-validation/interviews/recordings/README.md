# Interview recordings

Video files are **not committed to git** (size + privacy). Store externally and link here.

## Naming convention

```
YYYY-MM-DD_REGION_CODE_SalonName.ext
```

Examples:

- `2026-07-15_VN-01_LanNails.mp4`
- `2026-07-18_FI-03_KynsiStudio.mp4`
- `2026-07-20_INT-02_GlossBar.mp4`

## Storage

| Location | Purpose | Access |
|----------|---------|--------|
| Google Drive / Dropbox folder | Primary storage | Team only |
| Local backup | Redundancy | Encrypted disk |

**Folder link:** _Add shared drive URL when created_

## Consent

Every recording requires verbal consent at the start of the call:

> "This call will be recorded for internal research purposes only. The recording will not be shared publicly without your permission. Is that okay?"

Record consent confirmation in the transcript.
