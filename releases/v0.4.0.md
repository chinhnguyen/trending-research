# v0.4.0 — Vietnamese i18n and prompt review UX

**Date:** 2026-07-11

## Summary

Adds full **English/Vietnamese** UI and generation locale support, a **prompt review** flow before media generation, and smoother post-composer feedback while options are being created.

## Added

- **Locale switcher** — EN/VI toggle in the header; preference stored in the browser
- **Vietnamese translations** — Home, reports, briefs, prompts, sources, and post composer
- **Backend locale rules** — Research, briefs, and media prompt regeneration respect `preferred_locale`
- **Prompt review flow** — Edit hook, caption, hashtags, and generation prompt before accepting media
- **Pending option card** — Loading placeholder while a new post option is generated
- **Auto-generate countdown** — 10s timer starts only after the user edits or reloads a prompt field

## Changed

- Media generation prompts follow the selected UI language (including Vietnamese)
- Post setup picker and guidance panels use i18n strings
- AWS deployment passes media-generation env vars through Pulumi

## API / UI

| Change | Detail |
|--------|--------|
| `preferred_locale` | Accepted on research and brief generation requests |
| Media prompt API | Regeneration uses locale-aware system prompts |
| Web | `/briefs/:id` prompt review with inline accept and field reload |
